# PlaceholderAPI placeholders

# Hoe gebruik ik de PlaceholderAPI placeholders?

Om gebruik te maken van de PlaceholderAPI placeholders, moet je eerst PlaceholderAPI zelf installeren. Deze plugin kan je hier (opens new window) vinden. Als je deze plugin geïnstalleerd hebt en daarna je server herstart hebt, kun je de onderstaande placeholders gebruiken in elke plugin met PlaceholderAPI ondersteuning.

# Placeholders
